# paper

An archive of paper reading notes.

- One paper = one issue (issue title = paper title, labels = topics)
- Keep issues open
- Browse: https://hiroyukitanabe-21.github.io/paper/

## Setup

1. Place `index.html` and `.github/ISSUE_TEMPLATE/paper-note.yml` in this repository
2. Settings -> Pages -> Source: "Deploy from a branch" / `main` / `/ (root)`
3. Issues -> Labels: create topic labels (e.g. `optimization`, `llm`, `survey`)
4. Write notes via New issue and add topic labels for filtering
